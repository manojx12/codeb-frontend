# CodeB - MIS & Invoicing System (Frontend)

Angular frontend for the CodeB Internal Management System (IMS) — provides a UI for the Sales Team and HR to manage clients, estimates, invoices, payments, and HR operations.

## Tech Stack

- **Framework:** Angular
- **Styling:** Custom CSS (responsive design)
- **HTTP Client:** Angular HttpClient

## Modules

- **Dashboard** — summary stats across all modules
- **Client Management** — Clients, Chains, Brands, Subzones, Groups
- **Billing** — Estimates, Invoices, Payments
- **HR** — Employees, Attendance, Leaves, Payroll

## Setup Instructions

1. Clone the repository
```
   git clone https://github.com/manojx12/codeb-frontend.git
```

2. Install dependencies
```
   npm install
```

3. Make sure the backend is running at `http://localhost:8080` (see [codeb-backend](https://github.com/manojx12/codeb-backend))

4. Run the development server
```
   ng serve
```

5. Open `http://localhost:4200` in your browser
